RStudios GUI #1

RStudio

Et hurtigt overblik over RStudios brugerflade

Author

Søren O’Neill

Published

December 27, 2023

1 Et første overblik over RStudios GUI

RStudio er i udgangspunktet opdelt i 4 paneler (her illustrerert med forskellige farver). Dette layout kan ændres efter behov, men indtil videre, så hold dig til standard opsætningen.

Hjælp! Sådan ser mit layout ikke ud …

Hvis text editor feltet mangler, er det sandsynligvis bare fordi du ikke har noget dokument åben til at redigere i – prøv at klikke File / New file / R script (CTRL+SHIFT+n).

Hvis du ved et uheld har fået skjult et eller flere paneler, så prøv at klikke View / Panes / Show all panes (CTRL+SHIFT+o).

2 Ændre størrelse på panelerne

Klik på fanebladene herunder for at se tre forskellige måder at ændre på panelernes størrelser.

  • Klik-og-træk
  • Ikoner
  • Menu og tastaturgenvej

3 RStudio layout

Layout: forskellige farver, skrifttyper, tekst størrelse, osv)

Tools / Global options / Appearance … men indtil videre, lad de andre indstillinger være i fred, f.eks General, Code, etc.

4 Flere faneblade i hvert panel

Hvert panel i RStudio har som regel adskillige faneblade. For eksempel, i billedet herunder, indeholder ‘Output’ panelet fanebladene Files, Plots, Packages, Help, Viewer og Presentation.

Bemærk, at de faneblade som ses vil variere afhængigt hvad type dokument du arbejder med. Visse ekstra funktioner, som eksempelvis Git versionerings kontrol, vil også ændre på hvilke faneblade der præsenteres.

Øvelse #1

Prøv at gennemføre følgende:

  1. Lav tre faneblade i ‘Text editor’ panelet.
  2. Byt om på rækkefølgen af de tre faneblade i ‘Text editor’
  3. Brug hjælpefunktionen help til at afgøre hvordan du kan bruge R til at finde ud af hvilken version af R du har installeret på din computer.
  4. Find din R konsol. Tast: a <- 5 (tryk enter) – hvad skete der i dit Environment?
  5. Skift dit RSudio layout til et dark farve skema.
Svar på øvelse #1
  1. Lav blot tre nye tomme dokumenter at redigere: File / New file / R script (CTRL+SHIFT+n).
  2. Klik-hold-og-flyt med musen
  3. Du finder et faneblad ‘Help’ i Output panelet nederst til højre. Bemærk, at der er to søgefelter på help fanebladet – brug den til højre til at lære mere om (base)funktionen R.Version.
  4. Du finder konsolen _‘Console_’ i Input panelet nederst til venstre. Tilsvarende, i Environment fanebladet i Environment panelet, skulle du se en liste af ‘Values’ hvor ‘a’ er ‘5’.
  5. I Tools / Global options / Appearance, vælg f.eks ‘Cobalt’ i Editor theme dialogen.
